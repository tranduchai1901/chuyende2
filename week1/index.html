1. Bản mô tả các chức năng của ứng dụng

Đăng nhập & phân quyền cơ bản

  • Nhập tài khoản / mật khẩu, gọi API xác thực.
  • Sau đăng nhập: lưu thông tin học sinh (họ tên, lớp) để cá nhân hóa.

Bảng điều khiển (Dashboard)

  • Thống kê nhanh: số lần thi, điểm trung bình, điểm cao nhất.
  • Đề gợi ý theo lớp của học sinh.
  • Điều hướng tới các màn hình chính.

Khám phá đề thi

  • Chọn lớp, môn học.
  • Lọc độ khó (Dễ / Trung bình / Khó).
  • Tìm kiếm theo tên đề.

Làm bài trắc nghiệm

  • Hiển thị câu hỏi và các lựa chọn.
  • Thanh tiến độ theo số câu đã chọn.
  • Lưu đề yêu thích (tùy chọn).
  • Nộp bài, gửi đáp án lên server để chấm.

Kết quả & lịch sử

  • Màn kết quả: điểm %, số câu đúng / tổng số câu.
  • Lịch sử bài đã nộp theo thời gian.

Bảng xếp hạng & yêu thích

  • Xếp hạng theo lớp (điểm TB, số lần thi).
  • Danh sách đề đã đánh dấu yêu thích.

Backend & dữ liệu

  • API REST (Node.js + Express).
  • Lưu trữ SQLite: người dùng, lớp, môn, đề, câu hỏi, lượt thi, yêu thích.

3. Cách xem bằng Live Server (VS Code / Cursor)

  1. Mở thư mục dự án D:\ANDORID trong VS Code hoặc Cursor.
  2. Cài extension Live Server (Ritwick Dey) nếu chưa có.
  3. Chuột phải file week1/index.html → chọn Open with Live Server.
  4. Trình duyệt sẽ mở địa chỉ dạng http://127.0.0.1:5500/week1/index.html (cổng có thể khác).

Không bắt buộc Live Server: bạn cũng có thể mở trực tiếp file index.html bằng trình duyệt (double-click), nhưng một số trình duyệt hạn chế đường dẫn file — Live Server ổn định hơn cho bài nộp.
